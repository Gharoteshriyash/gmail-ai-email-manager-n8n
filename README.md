# AI-Powered Email Management Assistant

## Overview

This workflow automates email management using n8n, OpenAI, Gmail, and Google Calendar.

### Features

- AI email classification
- Automatic Gmail labeling
- AI-generated responses
- Google Calendar event creation
- Workflow automation

- Categorizing messages
- Drafting responses
- Scheduling meetings
